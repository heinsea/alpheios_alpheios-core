使用提示
  • 所有 mockup 都是单文件、纯静态、不依赖构建。需要的字体走 Google Fonts CDN，Tailwind 走官方 CDN。
  • 每个 mockup 顶部都使用同一份 token (CSS variables)。改色 / 改字号只改一处。
  • 主题切换只是预留：data-theme="dark" 写在 <html> 上即可，目前 mockup 默认 light。
  • 如果用 VS Code 的 Live Preview 或 IDE 内置预览看，右键文件 → "Open with Live Server" 体验更佳。
  • 详细规范见 ../DESIGN.md。
